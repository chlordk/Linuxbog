HTTP protokollen

HTTP står for Hyper Text Transfer Protocol, og HTTP er protokollen med hvilken information sendes over internettet. HTTP protokollen blev opfundet af Tim Berners-Lee fra MIT i 1994, HTTP version 1.0 er defineret i RFC 1945 dokumentet, og senere udvidet til HTTP version 1.1 der er dokumenteret i RFC 2068. Den mest basale HTTP funktion består i at downloade en side fra en webserver. Man kan emulere en browser ved at downloade en webside som følger:

telnet www.sslug.dk 80
GET / HTTP/1.0<enter><enter>

Webserveren vil så svare:

HTTP/1.1 200 OK
Date: Thu, 20 Jul 2000 16:02:01 GMT
Server: Apache/1.3.6 (Unix)  (Red Hat/Linux) PHP/3.0.7
Connection: close
Content-Type: text/html

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 3.2 Final//EN">
<HTML>
:
</HTML>

Dette er hvad browseren gør når den downloader en side fra en webserver. HTTP protokollen tillader også at man specificere hvilket sprog man fortrækker at modtage sider i, hvis serveren har sider i flere sprog så vil man modtage svar på det sprog man fortrækker. HTTP tillader autorization så sider kan beskyttes af passwords. En webserver kan også ligge cookies på en computer, små blokke af data der senere kan downloades. HTTP protokollen tillader også at man sender data tilbage til en webserver, fx. data indtastet i en form på nettet eller en fil.
